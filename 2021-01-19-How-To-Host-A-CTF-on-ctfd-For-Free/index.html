How To Host A CTF on ctfd For Free

2021-01-19

Hello Friends, In todays blog I will tell how to host CTF for free on the ctfd with the Digital Ocean.

Digital Ocean provide free 100$ Credit to every new user for one time.

Digital Ocean

We will use these credits for hosting our CTF.

NOTE : You will only get credits if you have credit card or paypal account.

But there is also one way by which you can take credits is that if you have debit card and have international ON on it then it will also work.

So after creating account you just have to go to home page and click on the Get started with a droplet.

After this you will get options to choose the OS you want the configuration you want and password you can set for the ssh login.

So from my experience ubuntu is good with 18.04* with configuration of

  • 4 CPUs
  • 5 TB Transfer
  • 160 GB SSD

Then click on the CREATE DROPLET option.

when the droplet is build you will get your IP.

Now login into machine you hosted with the SSH ( creds are the username : root and password : < you choosed while creating > )

Firstly update and upgrade

It will take sometime to update the system after this we have to install the docker and docker-compose.

After this just visit CTFD deployment website there also you get all the steps how to do.

CTFD DEPLOYMENT

Now just open the tmux session by typing tmux and press enter.

So we will git clone the ctfd repo first and then do the further steps.

Now move to the CTFD dir and open the docker-compose.yml file. In this we have to update the secret key and workers ( so that it can handel the load ).

Before :

After :

So now just save and exit the docker-compose.yml file. And type docker-compose up.

It will start your CTFD. It may take few minutes to complete the process so till then sit back and enjoy üòâ.

Now our CTFD is hosted at our IP know we have to point it to our subdomain. So I use cloudflare for the proxy server.

visit cloudflare website and there just click on the add site and choose free paln and after this your domain DNS recods wil be updated in the Cloudflare.

Now we just have to add our subdomain in the cloudflare and save it. Now our subdomain point to the IP address of our CTFD server.

After this just setup your admin account and Enjoy the CTF.

This is how you can host any ctf on the ctfd for free.

Extra Points

Sometimes it points the website but is not on SSL ( showing not secure ) for this solution just go to the cloudflare and visit ssl-tls and make the encryption mode flexible.

After this go to the edge-certificates.

Home -> ssl-tls -> edge-certificates

There you will find a option Always use HTTPS just on this option and after this your website will be on HTTPS.
