What is Memory Forensic ðŸ¤”

We all know when we start any program / application it first load in our volatitle memory i.e, RAM. Once system is shutdown than all the process in the RAM ( Volatile Memory ) are removed.
We know that some malware resides in the memory of the system and if any process is also started by the malware then we can find it in the memory of the system. So this is the best option to dump the memory of the affected system and check the volatile memeory of that system to detect suspicous process, files, even browser history and a lot of things

Now we know what is Memory Forensic we look how to do this with the FTK Imager Tool.

I am using sample of a memory dump used in TrollCat CTF to demostrate this.

After Downloading and installing the FTK imager we got a window of FTK Imager tool.

Go to file option and click on the add evidence option to add file in the FTK Imager to analyze it.

After importing file in the FTK Imager Toll on the left side i.e, EVIDENCE TREE we got all the Files, Dir detected by the Toll

Now start checking the Dir, Files loaded in the Evidence for the suspicous files. So if I continue with the same memory dump file so in this if we go to drive it have one zip TOPSECRET.ZIP.
Now we have to check what is inside this zip so for this RIGHT CLICK -> EXPORT FILES.

In this way we an search the complete files which were on the system at the time of dumping memory.
