How to install volatility in the kali-2020.0+

2021-03-15

Since volatility now ( Till 2021 ) needs python2

To install python2

sudo apt install python2

If you dont have python2 packages in the system then down the complete python2 from the Link

Then extract that and copy the python2.7 in

/usr/local/lib/

/home/$USER/.local/lib/

Then after that clone the volatility repo

git clone https://github.com/volatilityfoundation/volatility.git

Install the volatility

  • After you clone the repo move inside the directory

  • and run the command sudo python2 setup.py install
